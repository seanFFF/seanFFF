1. 1.website link Click here

2.I decided that the final website would allow users to interact with it, for example by clicking on the screen to bring up something. So I put this idea into practice, mainly by using the random command in p5js.

I wanted to interact with the screen by drawing circles with mouse clicks, so I set up these two commands first.

3.Afterwards, set up the draw circle command again, give the circle colours (hue, saturation, brightness, and opacity) via HSB, and set all four groups of colours to vary randomly between certain arrays, and combine all four colours together via the colour command.

4.Then set the coordinates and size of the circle, and also set it to random, after which the circle will appear by mouse click.

5.Since one circle at a time was a small number, I learnt a new command on the internet to make multiple circles appear at once, this command is a loop. So I set it up so that for i (which starts out as 0) when it is less than 5 i is added one at a time.

6.Finally I wanted to change the way the mouse would appear in circles of different shades when I clicked on different areas of the screen, so I linked the coordinates of the mouse to the shades for the purpose of a shade cycle.
